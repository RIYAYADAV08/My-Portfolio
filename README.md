# Personal Portfolio Website

A modern and responsive personal portfolio website built using React, Vite, Tailwind CSS, and Framer Motion.

## Features

* Responsive Design
* Modern UI & Animations
* About Me Section
* Technical Skills Showcase
* Projects Portfolio
* Experience Timeline
* Education Timeline
* Contact Section
* Resume Download

## Tech Stack

* React.js
* Vite
* Tailwind CSS
* Framer Motion
* React Icons
